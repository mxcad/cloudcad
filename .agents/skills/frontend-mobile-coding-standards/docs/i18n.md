# i18n 国际化

## 技术栈

VoerkaI18n — 与 PC 端（frontend）使用相同的 i18n 方案。

## 工作流

```
1. 提取中文文本 → pnpm i18n:extract
2. 翻译 → pnpm i18nAutoTranslate（自动）或手动
3. 编译 → pnpm i18n:compile
4. 构建 → pnpm build（自动包含 i18n 编译）
```

## 使用规则

### Vue 模板中

```vue
<template>
  <van-button>{{ t('按钮文本') }}</van-button>
</template>
```

### TypeScript 中

```typescript
import { useI18n } from 'vue-i18n';

const { t } = useI18n();
const label = t('文件名');
```

### 注意事项

- 所有用户可见的中文文本必须用 `t()` 包裹
- 提取后编译，编译后才能生效
- 不要手动编辑 `languages/*.js` 编译产物
- 不要手动编辑 `languages/*.ts` 源文件中的翻译（通过工具管理）

## 文件结构

```
src/languages/
├── zh.ts          # 中文源文件
├── en.ts          # 英文翻译
├── cht.ts         # 繁体中文翻译
├── index.ts       # i18n 初始化
├── storage.ts     # 语言存储
├── idMap.ts       # ID 映射
└── formatters/    # 格式化器
    ├── zh.ts
    ├── en.ts
    └── cht.ts
```

## 语言切换

```typescript
import { setCurrentLanguage } from '@/languages';

setCurrentLanguage('en');  // 切换到英文
setCurrentLanguage('zh');  // 切换到中文
```
