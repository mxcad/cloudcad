# 测试计划：资源库域

## 覆盖页面

| 页面 | 路由 | 组件 | 权限要求 |
|------|------|------|---------|
| 资源库（图纸库） | `/library`、`/library/drawing` | LibraryManager.tsx → ProjectDrawingsPanelMain (libraryType=drawing) | LIBRARY_DRAWING_MANAGE（ADMIN） |
| 资源库（图块库） | `/library/block` | LibraryManager.tsx → ProjectDrawingsPanelMain (libraryType=block) | LIBRARY_BLOCK_MANAGE（ADMIN） |
| 资源库详情 | `/library/:libraryType/:nodeId` | LibraryManager.tsx | 同上 |
| CAD 编辑器（从库打开） | `/cad-editor/:fileId?library=drawing` | CADEditorDirect.tsx | 需登录 |
| CAD 编辑器（从图块库打开） | `/cad-editor/:fileId?library=block` | CADEditorDirect.tsx | 需登录 |

## 执行注意事项

- 分页测试需确认 `pageSize=20` 是否与代码中一致
- 滚动加载阈值（500px from bottom）需要精确验证
- 上传后的 mxweb 转换依赖后端转换引擎，需等待转换完成
- 右键菜单在不同浏览器中行为可能不同（contextmenu vs custom）
- CAD 编辑器打开依赖 WebGL，需 headed 模式或 xvfb
- 权限测试需切换不同 storageState，建议使用 `test.use({ storageState: '...' })`
