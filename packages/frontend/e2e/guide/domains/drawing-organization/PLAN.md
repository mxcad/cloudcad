# 测试计划：图纸组织域

## 覆盖页面

| 页面 | 路由 | 组件 | 权限要求 |
|------|------|------|---------|
| 仪表盘（项目区） | `/dashboard` | Dashboard.tsx | 需登录 |
| 项目管理 | `/projects` | ProjectsPage.tsx | 需登录 |
| 项目文件管理 | `/projects/:projectId/files` | FileSystemManager/index.tsx (mode=project) | 需登录 + 项目成员 |
| 文件详情 | `/projects/:projectId/files/:nodeId` | FileSystemManager/index.tsx (mode=project) | 需登录 + 项目成员 |
| 私人空间 | `/personal-space`、`/personal-space/:nodeId` | FileSystemManager/index.tsx (mode=personal-space) | 需登录 |
