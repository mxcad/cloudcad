## 所需的种子数据

- [ ] 预建项目含 mxweb 文件（用于打开编辑）— 至少 2 个项目，各有 1+ mxweb 文件
- [ ] DWG/DXF 测试文件（用于上传转换）— 至少 1 个正常文件 + 1 个超大文件 + 1 个非支持格式文件
- [ ] 含外部参照的图纸 pair（主文件 + 参照文件）— 用于外部参照流程
- [ ] 各角色用户（OWNER, ADMIN, MEMBER, EDITOR, VIEWER）— 用于权限验证
- [ ] 资源库文件 — 用于 ?library=drawing 场景
- [ ] CAD 编辑器需 WebGL 环境 — CI 需 `--headed` 模式或 xvfb
- [ ] 后端 SVN 服务可用 — 保存/另存为依赖 SVN 提交

## 执行约束

- **fullyParallel: false** — CAD 编辑器 WebGL 单实例，图纸内容域内所有测试必须串行执行
- CAD 编辑器渲染到 `document.body` 级，需 `page.waitForSelector('#mx-cad-container')` 等待就绪
- Canvas/WebGL 内容是黑盒 — 仅测试 React DOM 外围（加载状态、弹窗、Toast、按钮可见性）
- 工具条按钮（保存/另存为/导出/撤销/重做）在 CAD UI（Vue/Vuetify）内部，使用对应的选择器策略
