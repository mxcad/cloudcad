# 图纸组织域 — 测试策略

## 页面清单

| 页面 | 路由 | 权限要求 | 说明 |
|------|------|---------|------|
| 项目管理 | `/projects` | 需登录 | 项目列表、私人空间入口 |
| 项目文件 | `/projects/:projectId/files` | 需登录 + 项目成员 | 文件树浏览、操作 |
| 文件详情 | `/projects/:projectId/files/:nodeId` | 需登录 + 项目成员 | 文件树含选中节点 |

| 仪表盘 | `/dashboard` | 需登录 | 项目卡片列表 |

## 核心工作流

### 项目管理流程
1. 创建项目 → 填写名称/描述 → 创建成功 → 自动创建默认文件夹
2. 编辑项目 → 修改名称/描述 → 保存
3. 删除项目 → 二次确认 → 级联删除文件
4. 归档项目 → 归档 → 只读 → 恢复
5. 项目列表 → 搜索 → 筛选 → 排序

### 文件树操作流程
1. 创建文件夹 → 输入名称 → 确认 → 树刷新
2. 创建文件（空白图纸） → 输入名称 → 创建 mxweb → 树刷新
3. 删除文件/文件夹 → 二次确认 → 移入回收站
4. 移动文件 → 选择目标 → 确认 → 树刷新 → 清除配额缓存
5. 复制文件 → 选择目标 → 确认 → 重名自动去重 → 树刷新
6. 批量操作 → 多选 → 批量移动/复制/删除
7. 拖拽 → 拖文件到目标文件夹 → 释放 → 移动

### 版本管理流程
1. 查看版本历史 → 版本列表 → 时间/操作人/备注
2. 版本对比 → 选择两个版本 → 查看差异
3. 版本恢复 → 选择版本 → 确认恢复

### 存储配额流程
1. 查看配额 → 进度条/已用/总量
2. 配额满 → 上传/保存被拒 → 提示

### 回收站流程
1. 查看回收站 → 已删除文件列表
2. 恢复文件 → 单文件/批量 → 恢复成功
3. 永久删除 → 单文件/批量 → 二次确认

## 交互覆盖要点

### Dashboard
- 项目卡片列表 → 分页/无限滚动
- 空状态：无项目时的引导提示
- 搜索框 → 输入 → 实时筛选
- 创建项目按钮 → 弹窗
- 加载状态：Skeleton cards

### FileSystemManager
- 顶部导航：项目名、面包屑
- 文件树 → 展开/折叠文件夹
- 文件列表 → 网格/列表视图切换
- 工具栏：新建、上传、搜索、排序、视图切换
- 批量操作栏（BatchActionsBar）：选中N个时显示 → 移动/复制/删除
- 右键菜单：打开、重命名、移动、复制、删除、版本历史
- 拖拽文件到文件夹 → 视觉反馈（高亮目标） → 释放移动
- 分页加载：向下/向上预加载阈值检测
- 空状态：空文件夹提示
- 过滤：按文件类型/状态 Tab 切换

### 弹窗
- 新建文件夹：输入名称 → 确认
- 重命名：输入新名称 → 回车确认
- 删除确认："确定删除？"
- 移动/复制：目标选择器（树形/搜索）
- 版本历史：时间线列表
- 属性：文件大小/格式/创建时间/修改时间

### 状态
- 加载中：文件列表 Skeleton
- 空：空文件夹引导文案
- 错误：网络错误重试按钮
- 成功：操作后 Toast "操作成功"
