# 系统管理域 — 测试策略

## 页面清单

| 页面 | 路由 | 权限要求 | 说明 |
|------|------|---------|------|
| 用户管理 | `/users` | SYSTEM_USER_READ | 用户 CRUD、搜索/筛选/排序/分页 |
| 角色管理 | `/roles` | SYSTEM_ROLE_READ | 角色 CRUD、权限分配 |
| 审计日志 | `/audit-logs` | SYSTEM_ADMIN | 操作日志浏览/筛选 |
| 系统监控 | `/system-monitor` | SYSTEM_MONITOR | 健康状态/缓存/性能监控 |
| 运行时配置 | `/runtime-config` | SYSTEM_CONFIG_READ | 系统配置读写 |

## 核心工作流

### 用户管理流程
1. 创建用户 → 填写信息 → 验证 → 创建 → Toast
2. 编辑用户 → 修改信息 → 保存 → Toast
3. 软删除用户 → 确认 → 移除 → 30天内可恢复
4. 立即注销 → 确认 → 永久删除
5. 恢复用户 → 管理员恢复 → 用户自助恢复
6. 搜索用户 → 输入 → 结果列表（用户名/邮箱）
7. 筛选 → 状态 Tab 切换：活跃/已注销
8. 排序 → 按创建时间/用户名
9. 分页 → 翻页控件
10. 存储配额管理 → 修改用户配额

### 角色管理流程
1. 查看角色列表 → 系统角色 + 自定义角色
2. 创建角色 → 名称 → 类别（系统/自定义） → 选择权限 → 确认
3. 编辑角色 → 修改名称/权限
4. 删除角色 → 二次确认
5. 权限分配 → checkbox 树（按类别分组）

### 审计日志流程
1. 浏览日志 → 时间线列表 → 分页
2. 筛选 → 按操作类型/用户/时间范围
3. 搜索 → 输入关键词

### 系统监控流程
1. 查看健康状态 → 后端/DB/Redis 状态
2. 查看缓存状态 → 命中率/键数量
3. 查看性能指标 → 响应时间/请求量

### 运行时配置流程
1. 查看配置列表 → 键值对 → 分组
2. 编辑配置 → 修改值 → 保存 → 生效
3. 新增配置 → 键名 → 值 → 类型 → 确认

## 交互覆盖要点

### UserManagement
- 搜索框 → 输入 → 实时/按回车搜索
- 状态 Tab：活跃/已注销
- 用户表格 → 排序（点击列头） → 分页
- 创建用户弹窗 → 表单：用户名/邮箱/密码/角色/配额
- 编辑用户弹窗 → 修改信息
- 删除确认弹窗 → "确定删除？"
- 存储配额弹窗（UserQuotaModal）
- 批量删除 → 选中多行 → 批量操作
- 空状态：搜索无结果
- 加载状态：表格 Skeleton
- 邮件/手机字段（mailEnabled/smsEnabled 运行时配置动态控制必填）
- 错误：创建失败（用户名重复/邮箱重复） → 错误提示

### RoleManagement
- 角色表格 → 角色名/类别/用户数
- 创建角色弹窗 → 名称/类别/权限 checkbox 树
- 权限分组：系统权限、项目权限、库权限
- 编辑角色 → 修改权限
- 删除角色 → 检查有关联用户 → 提示
- 权限树展开/折叠

### AuditLogPage
- 日志表格 → 时间/用户/操作/详情
- 筛选：操作类型下拉、用户搜索、时间范围选择器
- 分页

### SystemMonitorPage
- 健康状态指示器（绿/黄/红）
- 缓存面板 → 键数量/命中率图表
- 性能面板 → 响应时间图
- 自动刷新/手动刷新按钮

### RuntimeConfigPage (RuntimeConfigPage.tsx)
- 配置列表 → 分组显示
- 搜索配置 → 输入 → 筛选
- 编辑配置 → 行内编辑/弹窗编辑
- 配置值类型：string/number/boolean/json
- 保存 → Toast → 实时生效
- 敏感配置脱敏显示（password/api_key 类）
