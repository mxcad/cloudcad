# 图纸内容域 — 测试策略

## 页面清单

| 页面 | 路由 | 权限要求 | 说明 |
|------|------|---------|------|
| CAD 编辑器 | `/cad-editor`、`/cad-editor/:fileId` | 公开访问（CAD 引擎）；保存/导出需登录 | 全局覆盖层，React 外壳 |
| 仪表盘 | `/dashboard` | 需登录 | 项目列表、最近文件、存储统计 |

## 核心工作流

### 图纸上传转换流程
1. 上传 DWG/DXF → 后端转换 mxweb → 创建文件节点 → 文件列表刷新
2. 秒传（相同 hash） → 跳过上传 → 直接创建节点
3. 上传失败 → 格式不支持/文件过大 → 错误提示
4. 外部参照上传 → 检测缺失参照 → 上传替换文件 → 关联主文件

### 图纸打开流程
1. 打开图纸（从文件树点击） → 加载 mxweb → CAD 编辑器显示 → 加载中状态 → 渲染完成
2. 打开图纸（URL 直达 `?nodeId=xxx`） → 跳过欢迎面板 → 直接打开
3. 打开图纸（资源库 `/cad-editor/:fileId?library=drawing`） → 库文件打开
4. 打开图纸失败 → 401/404/网络错误/文件已删除 → 错误提示
5. 外部参照缺失 → 弹窗列出缺失项 → 上传替换后继续打开

### 图纸保存流程
1. 私人空间保存 → 原位覆盖 mxweb → 提交 SVN → 生成 bin
2. 项目保存 → 检查 CAD_SAVE 权限 → 原位覆盖 → SVN 提交
3. 资源库保存 → 检查库管理权限 → 直接覆盖 → 跳过 SVN
4. 另存为（项目/私人/资源库） → 选择目标 → 搜索 → 拷贝 → 不转换格式
5. 无权限保存 → 弹出"另存为"弹窗
6. 未登录保存 → 不可保存

### 图纸导出流程
1. 导出 DWG/DXF/PDF → 选择格式 → 后端生成 → 触发下载
2. 未登录导出 → 不可导出
3. 导出失败 → 格式不支持 → 错误提示

### 图纸关闭流程
1. 关闭/切换图纸 → 检查未保存修改 → 提示确认 → 操作

## 交互覆盖要点

### CAD 编辑器外围 React DOM
- 编辑器容器渲染：`#mx-cad-container` 存在
- 工具栏按钮：保存、另存为、导出、撤销、重做的 visible/enabled 状态
- 加载状态：CAD 引擎加载中 Spinner
- 网络错误：加载 mxweb 失败的错误提示

### "另存为"弹窗（SaveAsModal）
- 目标类型选择：私人空间/项目/资源库
- 目标搜索：输入 → 搜索结果列表
- 目标选择：点击 → 高亮
- 确认保存：按钮点击 → loading → Toast 成功
- 取消：X 按钮/点击遮罩/Esc
- 无权限目标：不可选择或 Toast 提示

### "导出"弹窗
- 格式选择：DWG/DXF/PDF 下拉
- 导出按钮：loading 状态
- 下载触发：导出成功后自动下载

### 外部参照弹窗（ExternalReferenceModal）
- 缺失参照列表：文件名列表
- 上传替换：单个/批量上传
- 关闭弹窗

### 未保存修改提示
- 弹窗：确认/取消
- 文本：描述未保存内容

## 种子数据要求

- 预建项目含 mxweb 文件（用于打开编辑）
- DWG/DXF 文件（用于上传转换）
- 各角色用户（用于权限验证）
- CAD 编辑器需 WebGL 环境，CI 需 --headed 模式或 xvfb

## 执行注意事项

- **`fullyParallel: false`**：CAD 编辑器 WebGL 单实例，图纸内容域内必须串行
- CAD 编辑器渲染到 `document.body` 级，需 `page.waitForSelector('#mx-cad-container')`
- 保存需后端 SVN 可用，CI 需预装 SVN 服务
- 外部参照测试需准备关联文件 pair（主文件 + 参照文件）
