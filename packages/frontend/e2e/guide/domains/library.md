# 资源库域 — 测试策略

## 页面清单

| 页面 | 路由 | 权限要求 | 说明 |
|------|------|---------|------|
| 资源库 | `/library`、`/library/:libraryType` | LIBRARY_DRAWING_MANAGE 或 LIBRARY_BLOCK_MANAGE | 图纸库/图块库管理 |
| 资源库详情 | `/library/:libraryType/:nodeId` | 同上 | 库文件详情/选中 |
| 字体库 | `/font-library` | SYSTEM_FONT_READ | 字体上传/下载/删除 |
| CAD 编辑器 | `/cad-editor/:fileId?library=drawing` | 需登录 | 从库打开图纸 |

## 核心工作流

### 图纸库流程
1. 浏览图纸库 → 分页加载（每页20条） → 向下/向上预加载
2. 搜索图纸 → 输入关键词 → 搜索结果
3. 打开图纸 → 点击 → 跳转 CAD 编辑器（`?library=drawing`）
4. 上传图纸 → 选文件 → 上传 → 转换 mxweb → 入库
5. 下载图纸 → 选择格式 → 后端转换 → 触发下载
6. 删除图纸 → 二次确认 → 移除
7. 移动/复制 → 选择目标文件夹 → 确认

### 图块库流程
- 与图纸库流程对称，`library=block`，权限为 LIBRARY_BLOCK_MANAGE

### 字体库流程
1. 浏览字体 → 字体列表 → 预览
2. 上传字体 → 选择字体文件 → 上传 → 验证格式 → 入库
3. 下载字体 → 点击下载
4. 删除字体 → 二次确认 → 移除

## 交互覆盖要点

### LibraryManager
- 库类型切换：图纸库/图块库
- 文件列表 → 网格视图
- 分页：向下滚动加载更多（距底部500px阈值）
- 搜索：输入 → 筛选
- 工具栏：上传、新建文件夹、删除
- 右键菜单：打开、移动、复制、下载、删除
- 面包屑导航
- 空状态：库为空提示
- 加载状态：Skeleton 加载

### LibraryManager — 特性
- 滚动位置恢复：向上翻页补偿 heightDiff
- 分页控件：首页/上一页/跳转/下一页/末页
- ProjectDrawingsPanelMain 组件（公用）：libraryType 区分图纸库/图块库
- 无版本管理（保存直接覆盖，不提交 SVN）

### FontLibrary
- 字体列表：字体名/格式/大小
- 上传弹窗 → 选文件 → 进度条
- 字体预览：点击字体 → 渲染示例文字
- 删除确认弹窗

### 状态
- 加载中
- 空库状态
- 上传失败（格式不支持/大小超限）
- 下载中
- 删除成功 Toast

| 操作 | ADMIN | USER_MANAGER | FONT_MANAGER | USER |
|------|-------|-------------|-------------|------|
| 浏览图纸库 | ✅ | ❌ | ❌ | ❌ |
| 上传图纸 | ✅ | ❌ | ❌ | ❌ |
| 下载图纸 | ✅ | ❌ | ❌ | ❌ |
| 删除图纸 | ✅ | ❌ | ❌ | ❌ |
| 浏览图块库 | ✅ | ❌ | ❌ | ❌ |
| 浏览字体库 | ✅ | ❌ | ✅ | ❌ |

### 无权限访问行为

| 场景 | 预期行为 |
|------|---------|
| 访问 /library（无权限） | NoPermissionPage |
| 访问 /font-library（无权限） | NoPermissionPage |
| 无权限用户看不到侧边栏入口 | 菜单隐藏 |

## 种子数据要求

- 图纸库预置 30+ 条 mxweb 文件（测试分页：2页以上数据）
- 图块库预置少量文件
- 字体库预置若干 ttf/otf 文件
- ADMIN 和 FONT_MANAGER 的 storageState
