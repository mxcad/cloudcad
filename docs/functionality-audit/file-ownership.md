# 文件所有权登记表

> agent 开始修改文件前必须在此登记，避免多个 agent 同时修改同一文件。
> 调度者在分配任务前检查此表，确保无冲突。

---

## 登记规则

1. agent 开始修改文件前，在此表中登记
2. 状态: `进行中` → `已完成` → `已释放`
3. 其他 agent 不可修改"进行中"的文件
4. agent 完成后将状态改为"已释放"，其他 agent 可以接管
