主题特性

🌓 系统主题跟随

自动检测系统主题偏好 (prefers-color-scheme)，无需手动设置即可自动切换明暗主题。

@media (prefers-color-scheme: dark) { ... }
♿ 减少动画偏好

尊重用户的减少动画偏好设置 (prefers-reduced-motion)，自动禁用过渡和动画效果。

@media (prefers-reduced-motion: reduce) { ... }
👆 触摸目标优化

所有交互元素最小点击区域 44×44px，符合 iOS HIG 和 Material Design 触摸目标规范。

min-height: 44px; min-width: 44px;
