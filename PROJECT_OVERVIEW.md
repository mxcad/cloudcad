# 项目概述

CloudCAD 是一个基于 Web 的 CAD 协作平台，支持在线编辑、版本控制和团队协作。

## 🎯 项目目标

- **在线 CAD 编辑** - 提供基于浏览器的 CAD 编辑能力，无需安装桌面软件
- **团队协作** - 支持多用户实时协作编辑和审阅
- **版本管理** - 集成 SVN 版本控制，追踪设计变更历史
- **文件管理** - 提供完整的项目、文件夹、文件管理体系
- **权限控制** - 基于角色的访问控制，确保数据安全

## ✨ 核心功能

### 1. CAD 编辑器

基于 mxcad 的 Web CAD 编辑器：

- **文件格式支持** - DWG、DXF 等 CAD 格式
- **矢量渲染** - 高性能 Canvas 渲染引擎
- **图层管理** - 图层显示/隐藏、锁定
- **视图操作** - 缩放、平移、旋转
- **测量工具** - 距离、角度、面积测量
- **标注工具** - 尺寸标注、文字标注

### 2. 文件系统

完整的文件管理体系：

- **项目管理** - 创建、编辑、删除项目
- **文件夹树** - 多级文件夹结构
- **文件操作** - 上传、下载、预览、删除
- **文件搜索** - 按名称、类型搜索
- **回收站** - 软删除恢复机制

### 3. 版本控制

集成 SVN 版本管理：

- **版本历史** - 查看文件所有版本
- **版本对比** - 对比不同版本差异
- **版本回滚** - 恢复到历史版本
- **版本标签** - 为版本添加标签
- **提交日志** - 查看版本提交记录

### 4. 权限管理

基于 RBAC 的权限系统：

- **角色管理** - 系统角色、自定义角色
- **权限分配** - 资源级、操作级权限
- **用户组** - 用户组权限继承
- **项目权限** - 项目级权限控制
- **审计日志** - 操作记录追踪

### 5. 团队协作

多用户协作功能：

- **实时协作** - 多用户同时编辑
- **评论批注** - 文件评论、批注
- **分享功能** - 生成分享链接
- **通知系统** - 变更通知、@提醒
- **在线审阅** - 在线审阅批注
