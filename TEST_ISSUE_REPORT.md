### 1. CAD 编辑器页面 JavaScript 错误（严重）

**问题描述**：访问 `/cad-editor` 页面时，前端报 JavaScript 运行时错误，导致页面无法正常加载。

**错误信息**：
- 错误类型 1：`setNodes is not defined`
- 错误类型 2：`setCurrentPage is not a function`

**页面状态**：页面显示"应用发生错误"提示，提供"重试"按钮，但点击后问题依旧存在。

**影响范围**：CAD 编辑器核心功能完全无法使用。
